clj-coffeescript by tbatchelli

A lean wrapper for rhino that allows to run javascript, coffeescript, and any code that would otherwise run in the broswer via env-js. JQuery, underscore.js etc... it's all fair game here

Authors

tbatchelli (tbatchelli@acm.org)

Contact

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/tbatchelli/clj-coffeescript

get the source code on GitHub : tbatchelli/clj-coffeescript
